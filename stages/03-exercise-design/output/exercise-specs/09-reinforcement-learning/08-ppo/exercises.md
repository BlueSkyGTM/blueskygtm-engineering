# Exercises — Proximal Policy Optimization (PPO)

## Exercises

1. **Implement** the clipped surrogate objective $L^{\text{CLIP}}(\theta)$ as a pure Python function. Your function should accept three arguments — `ratios` (list of floats), `advantages` (list of floats), and `epsilon` (float, default `0.2`) — and return the mean clipped surrogate loss. Test it on the following batch: ratios `[0.5, 0.9, 1.0, 1.1, 1.3, 2.0]`, advantages `[1.0, -0.5, 0.8, -0.3, 0.6, -1.0]`. Print the per-sample clipped values and the final scalar loss for `epsilon=0.1`, `epsilon=0.2`, and `epsilon=0.3`. Confirm which samples are clipped at each setting and explain in a printed summary line why clipping activates on those samples.

2. **Compute** probability ratios for a toy policy on a discrete action space. Define an old policy as a categorical distribution over 4 actions with logits `[0.0, 1.0, 2.0, 0.5]` and a new policy with logits `[0.5, 0.8, 2.5, 0.3]`. Convert logits to probabilities via softmax, then compute $r_t$ for each action. Print a table showing each action's old probability, new probability, ratio, and whether the ratio falls inside or outside the clip range $[0.8, 1.2]$. Then run the same computation after nudging the new logits by a large delta (`[3.0, 0.1, 5.0, -2.0]`) and observe how many ratios blow past the clip boundary. Print the count of clipped actions at each epsilon threshold from `0.1` to `0.5` in increments of `0.1`.

3. **Build** a multi-epoch PPO minibatch trainer on a single pre-collected rollout. Generate a synthetic rollout of 512 samples: each sample has an old log-prob (drawn from a standard normal), a continuous advantage estimate (drawn from `N(0, 1)`), and a placeholder for the current log-prob (initialized equal to old log-prob). Implement a training loop that runs 10 epochs of minibatch SGD (batch size 64), where each step perturbs the current log-prob by a gradient-norm-bounded delta and recomputes the ratio. After each epoch, print the epoch number, mean ratio, min ratio, max ratio, and the fraction of samples where the ratio was clipped. Your output should demonstrate that ratios stay bounded within $[1-\epsilon, 1+\epsilon]$ across all 10 epochs — if they don't, debug your clip application. Print a final summary line confirming whether clipping held.

4. **Implement** a TRPO-style KL-divergence penalty objective on the same synthetic rollout data from Exercise 3 (or regenerate equivalent data). Instead of clipping the ratio, compute the per-sample KL divergence between old and new policy distributions (you may approximate with the univariate Gaussian KL assuming unit variance). Add a penalty term $\beta \cdot D_{\text{KL}}$ to the surrogate objective with $\beta = 0.01$ and $\beta = 1.0$. Run 10 epochs of updates identical to Exercise 3 but using the KL-penalized objective instead of clipping. After each epoch, print the mean ratio, mean KL divergence, and the penalized loss. At the end, print a side-by-side comparison table: PPO clip (from Exercise 3) vs TRPO KL penalty at each $\beta$ — showing max ratio reached and max KL reached per method. State in a printed summary which method enforces tighter divergence bounds and at what computational cost.
