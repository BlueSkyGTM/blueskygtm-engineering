## Exercises

1. **Implement** a QMIX monotonic mixing network that takes a list of per-agent Q-values and a global state, produces `Q_tot`, and asserts that every weight emitted by the hypernetwork is non-negative. Seed the network with `torch.manual_seed(0)`, feed it 3 agents with random Q-values in `[0, 1]` and a 16-dim global state, and print the mixing weights alongside `Q_tot`. Confirm via assertion that `∂Q_tot/∂Q_i ≥ 0` for every agent by autograd-checking the gradient sign.

2. **Build** a MADDPG centralized critic that accepts a batch of joint observations and joint actions for `n_agents = 3`. Construct a simulated batch of 64 transitions (random tensors), forward-pass through the critic, and compute the TD loss `MSE(Q(s, a) − (r + γ · Q(s', a')))`. Print the scalar loss value and the gradient norm after one backward pass.

3. **Compute** GAE advantages for MAPPO using a global-state value critic. Generate a 128-step rollout with `γ = 0.99` and `λ = 0.95` from random reward and value tensors. Apply PPO clipping with `ε = 0.2` on ratio `π_new/π_old`, then compute and print the mean KL divergence between old and new policy logits (use random logits perturbed by Gaussian noise with σ = 0.1). Verify the KL is below a threshold you set.

4. **Compare** the three factorization approaches on a Matrix Game variant where two agents must select from 4 discrete actions each and the payoff matrix is structured so that the socially optimal joint action is individually suboptimal for each agent. Implement an independent DQN baseline, a QMIX learner, and a MADDPG learner over 2000 episodes. Plot the moving-average team reward for all three and print which method reaches 90% of the optimal joint payoff first (or note if none do).

5. **Design** and implement a CTDE coordination layer for a multi-agent GTM enrichment waterfall. Model three agents — a prospect-scoring agent, a Clay-enrichment agent, and an outreach-timing agent — each receiving only local context at execution time but trained centrally against a shared team reward (meeting-booked signal). Write `handlers/gtm_ctde_coordinator.py` that defines per-agent policy networks, a centralized critic over all agent contexts, a simulated training loop over 500 enrichment episodes with synthetic company data, and an execution-mode forward pass that runs each policy with local-only input. Save a plot of team reward curve to `outputs/skill-gtm-ctde-reward.png` and print the final per-agent action distributions.

6. **Prove or disprove** the following claim empirically and analytically: *"For any cooperative Markov game, there exists a monotonic mixing network (QMIX-style) that can represent the optimal joint action-value function."* Construct at least two counterexample environments where monotonicity prevents representing the optimal `Q*` (e.g., environments requiring negative credit assignment). Run QMIX and an unconstrained mixing baseline on each for 3000 episodes, print the gap to optimal, and write a 2–3 paragraph analytical argument to `outputs/skill-mixing-monotonicity-proof.md` explaining when the constraint helps (inductive bias, stabilization) and when it provably hurts.
