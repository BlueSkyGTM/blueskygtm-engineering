# Exercises — Monocular Depth & Geometry Estimation

## Exercises

1. **Run Depth Anything V3 on an image from the lesson and inspect the raw depth output.** Load the model, run inference on a single RGB image, and print the depth array's shape, dtype, min, max, mean, and standard deviation. Then manually select two pixel coordinates — one in a known foreground region and one in a known background region — and print their depth values. Verify that the foreground value is smaller (closer) than the background value, confirming the model preserves ordinal depth.

2. **Classify each production depth model by paradigm.** Write a Python script that prints a table to the terminal with columns: model name (MiDaS, Depth Anything V3, ZoeDepth, Marigold, Metric3D), paradigm (relative or metric), and training signal type (self-supervised, supervised relative, metric LiDAR ground truth, diffusion-based). Load any two of these models on the same input image and print their raw output ranges side-by-side. Confirm whether the ranges are comparable in scale or differ by orders of magnitude, and state which finding is consistent with each model's paradigm.

3. **Compute surface normals from a predicted depth map using finite differences.** Load a depth map produced by Depth Anything V3. For every interior pixel, approximate the local surface gradient using central differences in x and y, then compute the cross product of those gradient vectors to obtain a per-pixel surface normal. Normalize each normal to unit length. Print the percentage of pixels whose normals are within 20° of straight up (floor-like horizontal surfaces) versus within 20° of pointing toward the camera (wall-like vertical surfaces facing the viewer). Verify the percentages shift when you run the pipeline on an indoor room photo versus an outdoor landscape photo.

4. **Lift 2D bounding box detections into 3D space.** Run an off-the-shelf object detector (YOLOv8, Grounding DINO, or any detector of your choice) on an image to obtain at least three bounding boxes. Run Depth Anything V3 on the same image. Using a pinhole camera model with assumed intrinsics (fx=500, fy=500, cx=image_width/2, cy=image_height/2), back-project the center pixel of each bounding box into 3D camera coordinates. Print each detected object's class label, 2D center pixel, raw depth value, and resulting 3D (X, Y, Z) coordinate. Sort the objects from nearest to farthest and print the sorted list.
