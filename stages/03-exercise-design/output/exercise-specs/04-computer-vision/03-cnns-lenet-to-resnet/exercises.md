## Exercises

1. **Trace** the LeNet → AlexNet → VGG → Inception → ResNet lineage in a Python script. For each architecture, print its name, year, depth in layers, approximate parameter count, and the single mechanism it contributed (e.g., "ReLU + Dropout", "stacked 3×3 kernels"). Run the script and confirm the output lists five rows in chronological order with all five columns populated.

2. **Implement** a 2D convolution from scratch using only `numpy` — no `torch.nn.Conv2d`, no `scipy.signal`. Accept a `(1, 1, H, W)` input tensor, a `(out_channels, in_channels, kH, kW)` weight tensor, a stride, and a padding value. Feed a `5×5` input with a `3×3` kernel at stride 1, padding 0 and print the output. Then compute the receptive field by hand for a two-layer stack of `3×3` convolutions (stride 1, no padding) and verify your manual arithmetic matches the effective receptive field your function produces when you chain the two convolutions.

3. **Compute and compare** gradient magnitude through two 25-layer feedforward stacks: one using `torch.sigmoid` activations and one using `torch.relu` activations. Initialize each layer as a `nn.Linear(64, 64)` with default initialization. Pass a random input through the full forward pass, then backpropagate a dummy loss. For each stack, print the per-layer gradient norm (L2 magnitude of `.grad` on each layer's weight). Produce a `matplotlib` line chart with layer index on the x-axis and gradient norm on the y-axis, one line per activation. Save the plot to `outputs/activation-gradient-comparison.png` and print the ratio of the deepest layer's gradient norm to the shallowest layer's for each activation.

4. **Build** two 30-layer CNN blocks in PyTorch: one plain (conv → BN → ReLU repeated 30 times) and one residual (same layers but with a skip connection adding the block input to the block output). Feed a `(1, 64, 32, 32)` random tensor through each, backpropagate `output.sum()`, and print the gradient norm at layers 1, 10, 20, and 30 for both architectures. State in a printed summary which architecture maintains higher gradient magnitude at the deepest measured layer and by what factor.

5. **Extract visual firmographic embeddings from company website screenshots** using a frozen `torchvision.models.resnet50(pretrained=True)` as a feature extractor. Download homepage screenshots for 8 companies (use a tool like `playwright`, `screenshotone`, or manually collected PNGs in `data/screenshots/`). Strip the final FC layer, resize each screenshot to `224×224`, normalize with ImageNet statistics, and forward-pass each image through the frozen backbone. Save the resulting 2048-dimensional embedding vectors and the company names to `outputs/skill-visual-firmographic-embeddings.json`. Compute cosine similarity between all pairs and print a ranked list of the top 5 most similar company pairs — these are the firms your embedding model groups together based on visual design patterns. Write a one-paragraph analysis to `outputs/skill-visual-firmographic-embeddings.md` explaining whether the visual clustering aligns with any known firmographic attribute (industry, company size, target customer).
