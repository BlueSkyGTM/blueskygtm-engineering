## Exercises

1. **Run the pretrained Mask R-CNN and parse its outputs.** Load `torchvision.models.detection.maskrcnn_resnet50_fpn(pretrained=True)`, run inference on an image from `torchvision.datasets.CocoDetection` or any downloaded JPG with multiple objects, and print: number of detected instances, the class label and confidence score for each, bounding box coordinates, and the shape of each predicted mask tensor. Filter to only instances with score > 0.5. Your terminal output should list each detection as a numbered row.

2. **Trace feature map shapes through the backbone and FPN.** Instantiate the same pretrained model, access `.backbone` directly, and pass a `torch.randn(1, 3, 800, 800)` tensor through it. Print the shape of every feature pyramid level returned (the output is an `OrderedDict` keyed by `"0"`, `"1"`, `"2"`, `"3"`, `"pool"`). Then access `.rpn` and print the kernel sizes and strides of its convolution layers. Verify that the smallest FPN level has spatial dimensions near `200├Ś200` and the largest near `25├Ś25`.

3. **Implement RoIAlign from scratch using bilinear sampling.** Write a function `roi_align(feature_map, boxes, output_size=(7,7), spatial_scale=1/4)` that takes a single feature map tensor of shape `(1, C, H, W)` and a list of bounding boxes in image coordinates, then extracts each RoI by: (a) scaling boxes to feature-map coordinates, (b) dividing each RoI into a `7├Ś7` grid, (c) sampling the center of each cell via bilinear interpolation, and (d) stacking results into a `(N, C, 7, 7)` tensor. Compare your output against `torchvision.ops.roi_align` on the same inputs and print the maximum absolute difference. Also implement RoIPool (nearest-neighbor, no interpolation) and print the difference between RoIPool and RoIAlign on the same RoIs to demonstrate the quantization error that motivated the switch.

4. **Compare instance segmentation, object detection, and semantic segmentation on the same image.** Run three models on one image: `maskrcnn_resnet50_fpn` (instance segmentation), `fasterrcnn_resnet50_fpn` (object detection only), and `deeplabv3_resnet50` (semantic segmentation). For each model, print the top-5 predictions with scores. Then compute and print: (a) the total number of distinct object instances found by Mask R-CNN vs. Faster R-CNN, (b) the set of pixel-level class labels produced by DeepLabV3, and (c) for one shared class (e.g., "person"), how many separate masks Mask R-CNN produces vs. how DeepLabV3 treats it as one contiguous region. Your terminal output should make the three-way distinction visible.
