## Exercises

1. **Load and inspect a local image.** Use PIL to open any image on your machine (or generate one with `PIL.Image.new`) and convert it to a NumPy array. Print the `shape`, `dtype`, and a human-readable summary: height, width, channel count, and whether the image is grayscale, RGB, or RGBA. Run the script from your terminal and confirm the printed output matches what you expect when you open the file in an image viewer.

2. **Isolate and zero out the green channel.** Take an RGB image as a NumPy array, extract all three channels by slicing the last axis, set every value in the green channel to `0`, and recombine the three channels using `np.stack` or `np.concatenate`. Save the result with PIL and open it — the image should appear magenta-tinted. Print the mean of each channel before and after the modification to verify only green changed.

3. **Compute and compare per-channel statistics across color spaces.** Load a color image and convert it to grayscale and HSV using `colorsys`, PIL's `convert("L")`, or OpenCV's `cvtColor`. For each representation, print the min, max, mean, and standard deviation of every channel. Confirm that the grayscale version has exactly one channel and that the HSV version's Hue channel has a different value range than the RGB channels.

4. **Implement a thresholding function with a coverage metric.** Write `def threshold_mask(gray_array, threshold)` that accepts a 2D grayscale array and an integer threshold (0–255), returns a binary mask where pixels above the threshold are `255` and the rest are `0`, and prints the percentage of pixels that passed the threshold. Test it on at least two different images with two different threshold values each and confirm the percentage changes sensibly.

5. **Compare RGB vs. HSV for detecting a brand color in logos.** Download five company logos (e.g., from Clearbit Logo API at `https://logo.clearbit.com/{domain}` for domains like `stripe.com`, `spotify.com`, `slack.com`). For each logo, implement a color-detection mask in RGB (range check on R, G, B independently) and in HSV (range check on hue within a target band) for a specific target color such as Spotify green or Slack purple. For each logo and each method, print the pixel count classified as the target color. End with a one-line verdict: which space produced fewer false positives across the five logos. Save your script as `signals/examples/color_space_comparison.py` and ensure it runs end-to-end from the terminal, producing the printed comparison table.

6. **Build a CHW-conversion utility and prove it round-trips.** Write `def hwc_to_chw(arr)` and `def chw_to_hwc(arr)` that convert between HWC and CHW layouts using `np.transpose` or `np.moveaxis`. Load an RGB image, convert to CHW, print the shape to confirm the channel axis is now first, convert back to HWC, and verify with `np.array_equal` that the round-trip is lossless. Then load the image as a PyTorch tensor (or simulate with `torch.from_numpy`) and print the tensor's shape to confirm PyTorch expects CHW. Save the utility as `handlers/layout_utils.py` with a `__main__` block that demonstrates the round-trip on a sample image when run directly.
