# Exercises — Multi-Head Self-Attention

## Exercises

1. **Trace the shape transformations** through a multi-head attention forward pass. Create a random input tensor of shape `(2, 8, 64)` (batch=2, seq_len=8, d_model=64) with `num_heads=8`. Print the shape after each of the following stages: Q/K/V linear projection, reshape to `(B, T, h, d_k)`, transpose to `(B, h, T, d_k)`, the `QKᵀ` matmul, the `softmax` output, the attention-weighted `V` output, the transpose-back to `(B, T, h, d_k)`, the reshape to `(B, T, d_model)`, and the final output projection. Your terminal output should show nine shape lines, one per stage.

2. **Implement single-head scaled dot-product attention** from scratch using only `torch.nn.Linear` and `torch.nn.functional.softmax`. Given a random input `(1, 6, 32)`, project into Q, K, V each of dimension 32, compute `softmax(QKᵀ / √d_k) V`, and print the `6×6` attention weight matrix rounded to three decimal places. Verify that each row of the printed matrix sums to 1.0.

3. **Implement full multi-head attention** from raw linear projections with `d_model=64`, `num_heads=8`. Use the reshape-split-transpose pattern: project input to `3 × d_model` in a single linear layer, split into Q/K/V, reshape each to `(B, T, h, d_k)`, transpose, compute per-head attention, concatenate, and apply the output projection `W_O`. Run both your single-head implementation from Exercise 2 (adapted to `d_model=64`) and your multi-head implementation on the same random `(1, 10, 64)` input. Print the mean and standard deviation of each output tensor and the Jensen-Shannon divergence between their flattened output distributions.

4. **Sweep head count and measure attention pattern diversity.** For `d_model=128` and `num_heads ∈ {1, 2, 4, 8, 16, 32}`, initialize multi-head attention on a fixed random input `(1, 12, 128)` with a fixed manual seed. For each configuration, compute the average pairwise cosine similarity across all head attention weight matrices. Print a table with columns: `num_heads`, `d_k`, `avg_pairwise_cosine_sim`. Report which head count maximizes diversity (lowest cosine similarity) and which pattern you observe as `d_k` shrinks.

5. **Build a causal-masked multi-head attention module** and apply it to tokenized prospect data from a Clay or Apollo CSV export (columns: `company`, `title`, `persona_description`). Encode each row's concatenated text into a fixed-length token sequence using a simple bag-of-words or hash-based embedding into `d_model=128`. Run causal-masked multi-head attention with `num_heads=8` across the token dimension so each token attends only to earlier tokens (autoregressive mask). For 10 prospects, compute the attended representation for the final token position and print the top-3 prospects by highest attention-norm score. Write the full module and a short analysis of how masking changes the attention distribution compared to an unmasked run to `outputs/skill-causal-attention-prospects.md`.
