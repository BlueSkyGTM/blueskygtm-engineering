2. **Implement threshold-based routing.** Write a function `route_payload(classification_result, thresholds)` that accepts a classification output (predicted label + confidence) and a dictionary mapping each category to a minimum confidence threshold. Print a formatted table and identify the threshold where auto-route accuracy first exceeds 90%. State whether that threshold is practical given the resulting review burden.
