# Exercises — Singular Value Decomposition

## Exercises

1. **Factor and verify exact reconstruction.** Build a 5×6 term-document matrix from the lesson's corpus (or construct your own with integer counts). Compute the full SVD using `numpy.linalg.svd`, reconstruct **A** from **U**, **Σ**, and **V^T**, and print the Frobenius norm of the difference between the original and reconstructed matrix. Confirm the value is at machine-epsilon scale (≤ 1e-10).

2. **Truncate at rank *k* and confirm Eckart-Young.** Using the same matrix from Exercise 1, compute rank-2 and rank-3 truncated reconstructions. For each, print the Frobenius norm of the residual **A − A_k**. Then independently compute the sum of squares of the discarded singular values and print those. Confirm the two quantities match for both truncation levels.

3. **Build a latent similarity scorer for firmographic text.** Collect 8–12 company descriptions (paste them from Apollo, LinkedIn, or any public source). Vectorize them into a TF-IDF term-document matrix using `sklearn.feature_extraction.text.TfidfVectorizer`. Apply full SVD, keep the top 3 latent topics, and project every company into that 3-dimensional topic space. Define an ICP centroid as the mean vector of 2–3 companies you designate as ideal. Rank all companies by cosine similarity to the centroid and print the ranked list with scores.

4. **Compare full, truncated, and randomized SVD on compute time and singular value accuracy.** Generate a dense 500×200 random matrix. Time three approaches: `numpy.linalg.svd` (full), `sklearn.utils.extmath.randomized_svd` (n_components=20, 7 oversamples), and `scipy.sparse.linalg.svds` on the same matrix converted to sparse format. Print elapsed time for each and print the top 20 singular values from all three side by side. Report the maximum absolute difference between full-SVD singular values and each approximate method's values.

5. **Build a reusable prospect-rank handler that enriches and scores Clay-exported data.** Write a script that reads a CSV of company descriptions exported from Clay (or Apollo), vectorizes the text, applies truncated SVD at a configurable rank *k*, accepts a set of ICP seed-company names as CLI arguments, computes the centroid from those seeds, and writes a ranked CSV with similarity scores to `outputs/ranked_prospects.csv`. Accept `--rank` and `--top-n` flags. Print the top *n* prospects and their scores to the terminal. Persist the core scoring logic as a importable module at `handlers/svd_prospect_ranker.py`.

6. **Prove Eckart-Young empirically across random matrices and document the findings.** Write a script that generates 20 random matrices of varying dimensions and ranks (some low-rank with noise injected, some full-rank). For each matrix, truncate SVD at ranks 1 through *min(m,n)* and compute the residual error curve. Verify that the error at each rank *k* equals the sum of squared discarded singular values. Plot one representative error curve and save it to `outputs/skill-svd-eckart-young.md` along with a summary table of max deviation observed across all 20 trials, the matrix dimensions tested, and a one-paragraph explanation of why the theorem holds.
